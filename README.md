### 视频编辑接口示例 `/jove/?token=token&siteCode=siteCode&contentid=contentid`

#### 参数

| 名称      | 类型   | 说明            | 是否必传|
| ---------- | ------- | ------------- | ----|
| **token**     | string  | 用户的token    | 是|
| **siteCode**  | string | 站点的code    |   是|
| **contentid** | string | 素材的contentid|  是|
